## 🎯 なぜ私たちか

世の中の株アプリは、冷たいデータ端末かチャート描画だけのものばかり。私たちがやるのは「相場を見る」ことではなく、**AI に偉大な投資家のように考えさせる**こと：

- **9 人のマスター Agent がチャットに常駐**：バフェット / チャーリー・マンガー / ピーター・リンチ / ソロス / タレブ / リバモア / グレアム / フィッシャー + 現代価値分析 Agent —— 各自に理念ファイルとツール群
- **6 次元の価値投資フレームワーク**：評価 / 収益力 / 財務安全性 / キャッシュフロー / 成長性 / 配当の真実性 —— 銘柄の完全な価値ポートレートを一発で、ただ PE を見るだけじゃない
- **会話式インターフェース**：`@バフェット @グレアム 600519 を分析` で複数 Agent が同框、`/` で全スキル一覧、自然言語でバックテスト —— メニューの迷路なし

## ✨ 見どころ

- 💬 **@マスター + 自動データ取得**：メッセージに銘柄コードを書くだけで、プラットフォームが価値分析データを自動取得して LLM コンテキストに注入。マスター Agent が結論を出し、ツールが代わりに考えることはない
- 📊 **P/E P/B P/S のバリュエーションバンド**：パーセンタイル視点で割安かを判定、絶対数だけ見るのは終わり
- 📈 **K 線全量履歴 + 期間切替**：2010 年以降の全データ、日/週/月/四半期/年足をローカルで瞬時切替（pandas resample）
- 🏢 **同花順クローラーで会社情報**：本物の会社概要 / 業種 / セクターを取得、LLM の捏造なし
- 🧪 **ワンクリックバックテスト**：9 つの戦略テンプレート + 自然言語パラメータ + ウォークフォワード + モンテカルロ信頼区間
- 🐼 **純 pandas バックテストエンジン**：vectorbt 等の重依存なし、maverick-mcp (MIT) から移植、軽量で信頼できる
- 🌐 **中 / 英 / 日 / 韓 の 4 言語**：軽量な自前 i18n、localStorage で永続化

## 🧭 さらにできること

- 💼 **ポートフォリオ管理 + リスクダッシュボード**：VaR95/99、業種集中度、トレード前シミュレーション、レジーム別ポジション調整
- 🗣️ **マルチ Agent ディベート**：異なるスタイルのマスターに、一銘柄 / 一ポートフォリオで相互批判・応答・要約させる
- 🎯 **説明可能な 3 戦略スクリーニング**：bullish / bearish / supply_demand のスコアリング、ブラックボックスさよなら
- 🔌 **Agent ツール検索**：`GET /agent/tools` で全エンドポイント一覧、`GET /agent/web-search` でネット検索

## 🚀 3 ステップで開始

```bash
# バックエンド
cd backend && uv sync
cp config.example.yaml config.yaml
export LLM_API_KEY=your-key
uv run uvicorn app.main:app --reload --port 8000

# フロントエンド
cd frontend && npm install && npm run dev   # http://localhost:5173
```

フロントエンドを開き、チャットに `@バフェット 600519 を分析` と入力、あなたの価値投資の旅を始めよう。
