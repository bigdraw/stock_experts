## 为什么需要这个平台？

你有东方财富、同花顺、雪球——每个都能看行情。但当你想问"茅台现在值不值得买"时，得到的只是冰冷的 PE 数字。**你**变成了人肉分析师：在软件之间切换、手动算 ROE、把大把时间花在"帮自己分析"上。

> *"我不想自己算 PE 了。"*
> *"那我们建一个，让巴菲特替你算。"*

于是 9 位投资大师住进了对话窗口。每位都有自己的理念文件和可用工具——不是被分配的代号，是从真实的投资哲学中提取的：

- **巴菲特** — 买伟大企业，护城河+能力圈+安全边际+复利
- **芒格** — 多元思维模型+反向思考，合理价买好公司
- **格雷厄姆** — 安全边际+烟蒂股+市场先生，防御优先
- **费雪** — 成长股+闲聊法，合理价买卓越成长
- **彼得·林奇** — GARP+十倍股，投资你所了解的
- **索罗斯** — 反身性+可错性，止损快赢足
- **塔勒布** — 黑天鹅+反脆弱+杠铃策略
- **利弗莫尔** — 顺势+关键价突破，截短亏损
- **现代价值分析** — 6 维框架+分红真实性+现代思路

**Stock Analysis Platform** 是把孤立的行情数据变成投资决策的智能层——多会话、流式输出、上下文压缩、自动取数、6 维评估。

大多数软件帮你*看行情*。这个帮你*做决策*。

| 能力 | 说明 |
|------|------|
| **9 位投资大师 Agent** | 巴菲特/芒格/林奇/索罗斯/塔勒布/利弗莫尔/格雷厄姆/费雪+现代价值分析，每位有独立理念文件与可用工具 |

| **ChatGPT 式多会话** | 侧边栏会话管理、独立上下文记忆、**流式 SSE 输出**、**服务端上下文压缩**（参考 LobeChat 算法） |
| **@大师 + 自动取数** | `@巴菲特 分析 600519` → 自动拉取价值分析数据注入 LLM 上下文，大师据此下结论 |
| **历史 K 线 + 周期切换** | 2010 年至今全量数据，日/周/月/季/年K本地秒切，多数据源容错 |
| **P/E · P/B · P/S 估值带** | 百分位视角看贵不贵，均值+30/70分位带 |

| **组合风险看板** | VaR95/99、行业集中度、加仓前模拟、regime 调仓 |
| **Agent 工具发现** | `GET /agent/tools` 列出全部端点，`GET /agent/web-search` 联网搜索 |
| **多语言** | 中/英/日/韩 切换，轻量自建 i18n |

## 快速开始

### 一键启动

```bash
# Windows
scripts\start.bat          # 或 powershell -File scripts\start.ps1

# Linux / macOS
bash scripts/start.sh

# 停止
scripts\stop.bat           # Windows
bash scripts/stop.sh       # Linux / macOS
```

### 手动启动

```bash
# 后端
cd backend && uv sync
cp config.example.yaml config.yaml
export LLM_API_KEY=your-key
uv run uvicorn app.main:app --reload --port 8000

# 前端
cd frontend && npm install && npm run dev    # http://localhost:5173
```

打开前端，在对话里输入 `@巴菲特 分析 600519`，开始你的价值投资之旅。

**三层原则：**

| 层级 | 做什么 |
|------|-------|
| **大师 Agent 层** | 提供投资理念与分析框架，基于数据给出判断 |
| **数据层** | 提供行情/财报/K线/分红等事实，供大师分析 |
| **平台层** | 会话管理、上下文压缩、流式输出、agent 编排、工具发现 |

> *大师给判断，数据给事实，平台让它们相遇。*

## 6 维价值投资框架

调用 `GET /api/v1/stocks/{code}/value-analysis` 一次性获取：

| 维度 | 关键指标 | 判断标准 |
|------|---------|---------|
| **估值** | PE/PB/PS/PCF/Graham/FCF yield/股息率 | Graham 价 vs 现价、FCF yield >6% |
| **盈利能力** | ROE/ROA/ROIC/毛利率/净利率 | ROE>15%优秀，ROIC>WACC创造价值 |
| **财务安全** | 资产负债率/流动比/利息保障/现金比 | 负债率<40%安全，利息保障>5 |
| **现金流** | OCF/FCF/盈利质量(OCF÷净利) | 盈利质量>1.0真金白银 |
| **成长性** | 营收/净利/净资产 CAGR 3y/5y | 10%+稳健增长 |
| **分红真实性** | 股息率/派息率/连续年数/分红增速 | 稳定>3%且派息可持续 |

## 理念

### 工具不替大师下结论

所有分析工具只提供数据——`/agent/tools` 列出能力清单、`/chat/analyze/stock` 验证存在+取价值分析数据、`/agent/web-search` 联网搜索。**决策由大师 Agent 基于数据自行判断。**

> *工具给事实，大师给判断。* — 工具不思考，大师不取数。

### 五条第一性原理

| # | 原理 | 一句话 |
|---|------|-------|
| P1 | 数据驱动优先 | 所有判断基于量化数据，不靠"感觉" |
| P2 | 大师理念 + 工具数据 | 理念决定框架，数据决定结论 |
| P3 | 不瞎编 | 股票/基金不存在就说不存在，不幻觉 |
| P4 | 单一真相源 | LLM 配置 DB 存、回测纯 pandas、行情多源容错 |
| P5 | 可验证才算完成 | 测试通过、ruff clean、build 过 |
